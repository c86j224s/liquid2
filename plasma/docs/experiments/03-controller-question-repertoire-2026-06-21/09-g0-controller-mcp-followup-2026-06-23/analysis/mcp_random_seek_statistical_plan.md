# MCP Random-seek Statistical Plan

Design: paired mission-seed blocks with cells R0 and R1. First formal look
requires at least 10 complete clean blocks. Target is 14 blocks, capped at 18
unless a separate power review approves more.

Success requires R1 to improve or tie R0 on navigation efficiency, depth,
breadth, or provenance completeness without increasing unsupported or
unverifiable conclusions beyond the pre-registered non-inferiority threshold.

R1 is an access surface only. It must not duplicate source corpora or reclassify
generated artifacts as source material.
