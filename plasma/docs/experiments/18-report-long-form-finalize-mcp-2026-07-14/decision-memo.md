# Experiment 18 Decision Memo

Do not adopt the long-form finalization MCP boundary on the evidence from this
experiment.

The scoped product path passed the corrected smoke and 11 completed candidate
quality runs showed the required finalizer/canonical/sentinel linkage. However,
one of 12 candidate quality runs failed after the intention-to-treat boundary,
so the preregistered machine gate failed. The analysis controller also stopped
before confirmatory statistics because it omitted the existing intention-to-
treat score assembly call. The protocol requires adoption to satisfy both the
machine gate and the confirmatory quality gates; neither condition can be
claimed here.

The result is a stopped, non-adopting experiment, not evidence of quality
inferiority or superiority. Any successor would require a separately approved
protocol and controller lock; experiment 18 must not be overwritten or resumed
with changed analysis code.
